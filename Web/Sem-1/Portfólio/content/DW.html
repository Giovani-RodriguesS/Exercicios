HTML semântico, CSS e JS. Em HTML, conhecemos as principais tags usadas numa página, como span a, para links, p , para parágrafo, h1, para título de cabeçalho, entre outras. Exploramos ainda o uso de classes e IDs, que são úteis quando vamos estilizar a página e desenvolver interatividade com JavaScript.

Em CSS, vimos que podemos selecionar um elemento do documento HTML pela Tag, classe e ID. Pela tag, selecionará todos os elementos dessa tag. A vantagem e desvantagem de selecionar dessa forma é todos os elementos serão selecionados. A seleção pela classe, permitirá uma estilização mais seletiva, pois os elementos impactados serão aqueles que tiverem como atributo a classe selecionada. Já pelo ID, apenas um elemento será estilizado, visto que o ID é único, não podendo ser distribuído a outro elemento HTML.

Em JavaScript, aprendemos a sintaxe da linguagem e como manipular os elementos HTML pela árvore DOM. A árvore DOM é composta por nós que são os elementos da página HTML e por meio dela podemos acessar os elementos da página manipular seus comportamentos.
