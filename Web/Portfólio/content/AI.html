Arduino

No início do semestre, tivemos contato com o sistema embarcado Arduino. Para fins didáticos, usamos a plataforma TinkerCad, e nela aprendemos a utilizar alguns dispositivos controlados pelo embarcado, como o Buzzer, Led RGB e botões. Na programar usamos a linguagem C++.


Alguns conceitos abordados nas aulas foram:

    DigitalWrite() - Para saida de sinal do arduino para um pino específico;
    DigitalRead() - Para ler o estado de um pino digital;
    AnalogWrite() - Para saida analogica num pino específico;

Além desses, ainda tivemos contato com o monitor Serial, uma ferramenta útil para verificar: estado do botão, potência do potenciomêtro e até a temperatura de um sensor de temperatura.

    Serial.begin(9600) - Definimos no Setup para iniciarmos o monitor serial. Obs: 9600 é a taxa de transmissão.

Esp32

O microcontrolador ESP32 foi outro dispositivo que vimos nas aulas. Porém, mesmo sendo raso o contato que tivemos com ele, tivemos uma avaliação, onde desenvolvemos um programa que imprimia no display do LCD as opções (contar toques, relógio e zerar dados).


Alguns conceitos abordados nas aulas foram:

    LCD setCursor() - Para definir a posição do cursor no display;
    LCD clear() - Para limpar todas as informações do display;
    LCD print() - Para inserir informações no display;
